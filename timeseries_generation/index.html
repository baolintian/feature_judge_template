JSON format

The format of the data when exported in the JSON format is the following 

{
 'times': [0.0, 0.1, ...],
 'values': [12.5, 13.4, ...]
}

CSV format

The format of the data when exported in the CSV format is the following 

time,value
0.0,12.5
0.1,13.4
...
